# Device to Device

#### Device to Device - protocol id: 0

| 0   | Heart beat                                              |     |
| --- | ------------------------------------------------------- | --- |
| 1   | Error                                                   |     |
| 2   | Hello. Allows for devices to ch*ec*k if they are paired |     |
| 3   | Pair Request                                            |     |
| 4   | Send Key                                                |     |
| 5   | Key Check                                               |     |
| 6   | Key Check Response                                      |     |
